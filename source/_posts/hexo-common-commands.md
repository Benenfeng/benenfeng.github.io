#### hexo new
```bash
hexo new demo # 生成demo 
```

#### hexo g
``` bash
hexo g #生成静态文件
```

#### hexo s
```bash
hexo s # 本地开启hexo服务 默认端口4000
```

#### hexo d
```bash
hexo d #部署到远端
```

#### hexo clean
```bash
hexo clean # 清除生成的静态文件
```
